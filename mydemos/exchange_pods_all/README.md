用法：

该脚本主要用于在k8s集群中，将一套环境的所有镜像获取出来，替换到另外一套环境的服务中
1. 使用时，依据当前版本环境的具体参数修改配置信息
2. 配置信息中，base_path为各服务公共目录的最深一层，log_file为日志文件
3. 使用前必须保证各个服务deploy文件的上一层目录名与镜像服务名一致
    如: /data/…/wbs-web/deploy.yml   ==>   wbs-web:1234567890
4. 将该脚本复制到服务器中使用：
    1）在镜像版本比较新的服务器，执行‘grep -r "image: registry.newbanker.cn:5000/newbanker/" ./ > image.txt’，
获取到包含待更新的镜像版本的image文件
    2）将image.txt文件拷贝到目标服务器下，该脚本相同目录下
    3）执行该脚本
